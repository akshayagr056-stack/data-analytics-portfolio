# Superstore Sales Analysis

## Project Overview

This project analyzes Superstore sales data to identify trends, profitability drivers, and opportunities for business growth.

## Business Question

How can the company improve sales and profitability using historical sales data?

## Objectives

* Analyze sales performance
* Identify profitable categories and regions
* Understand customer segments
* Provide actionable business recommendations

## Tools Used

* Python
* Pandas
* Jupyter Notebook
* SQL
* PowerPoint

## Data Cleaning

* Checked missing values
* Verified duplicate records
* Examined data types
* Prepared data for analysis

## Analysis Performed

* Sales by Category
* Profit by Region
* Sales by Segment
* Business Performance Analysis

## Key Findings

* Identified top-performing categories
* Evaluated regional profitability
* Analyzed customer segment contribution
* Discovered business improvement opportunities

## Recommendations

1. Focus marketing efforts on high-performing categories.
2. Improve profitability in low-margin business areas.
3. Expand successful regional strategies.
4. Use data-driven decision-making for future growth.
